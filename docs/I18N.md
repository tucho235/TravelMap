# Sistema Multi-Idioma (i18n) – TravelMap

TravelMap soporta múltiples idiomas en PHP y JavaScript. Los archivos de traducción son JSON independientes por idioma.

## Idiomas disponibles

| Código | Idioma |
|---|---|
| `en` | English (por defecto) |
| `es` | Español |

El idioma por defecto del sitio se configura en **Admin → Configuración → Idioma por defecto**.

Los usuarios pueden cambiarlo en cualquier momento desde el selector del panel lateral. La preferencia se guarda en `localStorage`.

---

## Archivos de traducción

```
lang/
  en.json    ← Inglés
  es.json    ← Español
```

Las claves están organizadas jerárquicamente y se acceden con notación de punto:

```json
{
  "app": { "name": "TravelMap" },
  "navigation": { "home": "Home", "trips": "Trips" },
  "map": { "my_trips": "My Trips" }
}
```

---

## Uso en PHP

```php
echo __('app.title');
echo __('navigation.home');
```

La función `__($key, $default)` está definida en `src/helpers/Language.php`.

---

## Uso en JavaScript

```javascript
const title = __('map.my_trips');
```

La función está definida en `assets/js/i18n.js`.

---

## Agregar un nuevo idioma

1. Copiar `lang/en.json` como `lang/fr.json` (u otro código)
2. Traducir todos los valores del nuevo archivo
3. En `src/helpers/Language.php`, añadir el código al array `$availableLanguages`
4. En `assets/js/i18n.js`, añadir el código al array `availableLanguages` y el nombre en `getLanguageName`
5. En `index.php`, añadir la opción al selector de idioma
6. En `admin/settings.php`, añadir el idioma al listado de opciones

---

## Agregar nuevas traducciones

Al añadir texto visible al usuario, **nunca hardcodear el texto** directamente en el código. Siempre:

1. Agregar la clave en `lang/en.json` y en todos los demás archivos de idioma
2. Usar la función helper correspondiente (`__()` en PHP o JS)

**Prioridad de detección de idioma** (de mayor a menor):
1. Parámetro GET `?lang=xx` (PHP) / `localStorage` (JS)
2. Cookie `travelmap_lang`
3. Configuración del sistema en base de datos
4. Cabecera `Accept-Language` del navegador
5. Inglés (fallback)
